# Moving Reference Frame in Unity 6.2 Demonstration Software

Intended to work for walking around inside a moving vehicle or similar.


## Instructions for Build and Use

Steps to build and/or run the software:

1. Download Unity 6.2 Game Engine (via Unity Hub)
2. Download this entire project folder
3. Open it using Unity Hub GUI

Instructions for using the software:

1. Press the top-center play button. Wait for the software to build.
2. Use cursor and WASD arrow keys to look and move around. Press and hold SPACEBAR to apply an upward force.
